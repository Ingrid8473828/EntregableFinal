En el modelo físico implementado, se simula el campo eléctricamente generado por dos placas que tienen carga opuesta, es decir, una carga positiva y una carga negativa, las cuales discretizamos en n cargas puntuales, las cuales se distribuyen a lo largo de su longitud. A la hora de calcular la determinación del campo eléctrico, utilizamos la ley de Coulomb, $\\vec{E} = k\_e \\frac{q}{r^2} \\hat{r}$ y aplicamos el principio de la superposición por adicionar todas las contribuciones a partir de cada dos cargas. En el código, llegamos a las expresiones $E\_x = \\frac{qx}{r^3}$ y $E\_y = \\frac{qy}{r^3}$, donde $r = \\sqrt{x^2+y^2+\\epsilon}$ hemos añadido un término pequeño para prevenir las singularidades numéricas. Se incorpora también el cálculo del potencial eléctrico mediante $V = k\_e \\frac{q}{r}$ ya que de esa forma podemos mostrar las líneas del campo eléctrico y las superficies equipotenciales. Estas representaciones muestran cómo el campo eléctrico se da desde la placa positiva a la negativa, y cómo la intensidad de dicho campo eléctrico disminuye conforme aumenta la distancia a la que se esté.

A continuación, simulamos el movimiento de las partículas (glóbulos) en tal campo, a cada partícula le corresponde una carga de magnitud q y una masa de cantidad m, lo que da lugar a la fuerza eléctrica que tiene que sufrir la partícula mediante $\\vec{F} = q\\vec{E}$ o, de forma alternativa, utilizando directamente la ley de Coulomb entre la partícula y cada elemento de carga; $\\vec{F} = k\_e \\frac{q dq}{r^2} \\hat{r}$. A partir de tal fuerza consideramos la segunda ley de Newton $\\vec{F} = m\\vec{a}$, de la que inferimos la aceleración, para continuar resumiendo el movimiento mediante integración numérica aplicando el método de Euler: $\\vec{v} = \\vec{v} + \\vec{a}dt$ y la posición de la partícula mediante la fórmula $\\vec{x} = \\vec{x} + \\vec{v}dt$. Este proceso se repite de forma iterativa hasta que la partícula alcanza cierta posición o bien hasta llevar a cabo un límite de iteraciones. Observamos que las partículas que tienen mayor masa (glóbulos parasitados) se caracterizan por tener menor aceleración de la misma fuerza, lo que cambia las trayectorias y el tiempo de desplazamiento. Estas diferencias físicas generan patrones de las variables de posición final o el número de iteraciones, que luego se utilizan como entrada para los algoritmos de clasificación de inteligencia artificial.
